# vision-widget

항상 화면 위에 떠 있는 **Personal Command Center** 데스크탑 위젯 (Tauri v2 + React 18).

전통적 비전보드(정적 이미지 콜라주)가 아니라 **살아있는 화이트보드** — 영감 문구·단계별 목표(비전보드 요소)와 실시간 프로젝트 현황·GitHub 연동·습관 추적·포모도로 타이머(대시보드/생산성 요소)가 공존한다. 우주선 기장이 계기판을 보듯 삶 전체를 한눈에 통제하는 감각을 목표로 한다.
