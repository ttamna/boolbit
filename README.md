# Vision Widget

투명 데스크탑 위젯 — Tauri v2 + React + TypeScript

항상 위(always-on-top) + 투명 배경 + 드래그 이동 가능한 개인 대시보드.

---

## 기능 목록

우선순위 순으로 정렬. ✅ = 구현 완료, 🔲 = 로드맵.

### 1. 핵심 위젯

| 우선순위 | 기능 | 설명 |
|----------|------|------|
| ✅ 1 | **항상 위 투명 창** | always-on-top + 배경 투명 + 윈도우 크롬 없음 + 작업표시줄 숨김 |
| ✅ 2 | **실시간 시계** | 시:분:초 + 한국어 요일/날짜, 1초 갱신 |
| ✅ 3 | **프로젝트 카드** | 상태(active/in-progress/paused) + 진행률 바 + 목표 + 커스텀 지표 |
| ✅ 4 | **습관 스트릭 트래커** | 이모지 아이콘 + 연속 일수 + 클릭 +1 + 색상 강도(10일↑ 초록) |
| ✅ 5 | **포모도로 타이머** | 25분 집중 / 5분 휴식 + OS 알림 + 진행률 바 + 접기/펼치기 |
| ✅ 6 | **영감 문구 로테이션** | 8초마다 페이드 전환, 자동 루프 |

### 2. 인터랙션 & 편집

| 우선순위 | 기능 | 설명 |
|----------|------|------|
| ✅ 7 | **인라인 편집** | 프로젝트명·목표·지표값 클릭 → 즉시 편집, Enter/Escape 처리 |
| ✅ 8 | **습관 스트릭 클릭 증가** | 숫자 클릭 → +1, 즉시 저장 |
| ✅ 9 | **습관 이름 편집** | 습관 이름 클릭 → 인라인 편집 |

### 3. 창 관리

| 우선순위 | 기능 | 설명 |
|----------|------|------|
| ✅ 10 | **드래그 이동** | 상단 드래그 바로 네이티브 창 이동 |
| ✅ 11 | **위치 프리셋 (4 코너)** | ◤◥◣◢ 버튼으로 화면 모서리 스냅 (여백 20px) |
| ✅ 12 | **위치 자동 저장/복원** | 앱 재시작 시 마지막 위치 복원 |
| ✅ 13 | **콘텐츠 높이 자동 리사이즈** | ResizeObserver로 내용 크기에 맞게 창 높이 자동 조정 |

### 4. 시스템 통합

| 우선순위 | 기능 | 설명 |
|----------|------|------|
| ✅ 14 | **시스템 트레이 아이콘** | 표시/숨김 토글 + 종료 메뉴 |
| ✅ 15 | **닫기 → 트레이 최소화** | X 버튼이 앱 종료 대신 트레이로 숨김 |
| ✅ 16 | **포모도로 OS 알림** | 집중/휴식 완료 시 데스크탑 알림 (`tauri-plugin-notification`) |

### 5. 설정 & 커스터마이징

| 우선순위 | 기능 | 설명 |
|----------|------|------|
| ✅ 17 | **투명도 조절 슬라이더** | 20~100% 범위, 실시간 적용, 저장 |
| ✅ 18 | **설정 패널 (접기/펼치기)** | ⚙ 아이콘으로 토글, 현재 불투명도 표시 |

### 6. 데이터 & 영속성

| 우선순위 | 기능 | 설명 |
|----------|------|------|

| ✅ 20 | **설정 JSON 자동 저장** | 위치·크기·투명도 → `vision-widget-settings.json` |

### 7. 로드맵

| 우선순위 | 기능 | 설명 |
|----------|------|------|
| 🔲 21 | **GitHub API 연동** | 커밋 수·PR 수 자동 업데이트 (T4-1) |

## 데이터 저장 경로

- **Windows**: `%APPDATA%/vision-widget-data.json` (콘텐츠), `%APPDATA%/vision-widget-settings.json` (설정)
- **macOS/Linux**: `~/.config/vision-widget-*.json`

## 사전 준비

```bash
# 1. Rust 설치
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

# 2. Tauri v2 필수 의존성 (Windows)
# → https://v2.tauri.app/start/prerequisites/

# 3. Node.js 18+
```

## 설치 & 실행

```bash
npm install
npm run tauri dev    # 개발 모드 (hot reload)
npm run tauri build  # 프로덕션 빌드
```

## 커스텀

### 프로젝트·습관·문구 수정

### 포모도로 시간 변경

`src/components/PomodoroTimer.tsx`의 `FOCUS_MINUTES`, `BREAK_MINUTES` 상수 수정.
